# Odorata Restaurant

![Screenshot](/public/images/screenshot.png)

A minimal restaurant website built with Webpack, npm, ES6 modules, and vanilla JavaScript.


## Prerequisites

Before setting up the project locally, make sure you have the following installed:

- Node.js (v14 or higher recommended)
- npm (comes with Node.js)

## Local Setup

### 1. Clone the repository

```bash
git clone https://github.com/melio0504/restaurant-page.git
cd restaurant-page
```

### 2. Install dependencies

Install all required npm packages:

```bash
npm install
```

### 3. Development

To run the project in development mode with hot-reloading:

```bash
npx webpack serve
```

This will start the webpack dev server. Open your browser and navigate to the URL shown in the terminal (typically `http://localhost:8080`).
